MODEL COMPARISON — EXPLANATION

Based on the evaluation results, the Polynomial Regression model achieved the best performance among the three models tested. It obtained the highest R² score (0.958) and the lowest error values (MAE, MSE, and RMSE), indicating better predictive accuracy.

This suggests that the relationship between the input variables and the target variable is non-linear. While Linear Regression assumes a straight-line relationship, Polynomial Regression is able to capture more complex patterns in the data, which explains its superior performance.

The Neural Network (ANN) also performed well, but it did not outperform the Polynomial model. This may be due to limited data or the need for further hyperparameter tuning.

In conclusion, Polynomial Regression is the most appropriate model for this dataset because it provides the best balance between accuracy and model complexity.
